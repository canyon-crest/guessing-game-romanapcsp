# Above and Beyond
  
## Extra features used:

* ### Streaks:
    *  #### Increments by one each win, if they give up resets.

        Line 7 defines the variable for use.
        
        Line 97-104 creates the function in use and has an if statement adds one to the variable each time or resets to zero if user gives up.

        #### This improves the game because many popular apps such as Duolingo use streaks as an addictive manner to get users using their application more often.


* ### Dyamic Background 
    * #### Added dynamic background that changes depending on how cold, warm, or hot the user is.

    Turns blue: when cold

    Turns orange: when orange

    Turns yellow: when rights
    
    Turns red: when warm

    Code is found from line 60-85 in the makeGuess function where the background color activates depending on the difference between the users guess and the correct answer.

    #### This addition to the guessing game adds a layer of fun and creativity to the game, making the user more encouraged to get the correct guess and helps the user in an instant now how close they are without having to read the small text.
